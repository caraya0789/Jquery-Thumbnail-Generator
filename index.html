Jquery Thumbnail Generator

This plugin resizes the image proportionally so it doesn't loose quality showing as much as possible of the image inside a generated wrapper which "crops" the image

Parameters

Option Type Default
width number 100
height number 100
aligh string "middle"

How to Use

Basic Usage


$('img').thumbnail();

<img src="path/to/image.jpg" />

Example

Setting Default Parameters


$('img').thumbnail({
    width:150,
    height:150,
    align:"middle"
});

<img src="path/to/image.jpg" />

Example

Setting Parameters on single image


$('img').thumbnail();

<img src="path/to/image.jpg" data-width="200" data-height="300" data-align="middle" />

Example

Different align values


$('img').thumbnail();

<img src="path/to/image.jpg" data-align="topleft" />
<img src="path/to/image.jpg" data-align="middle" />
<img src="path/to/image.jpg" data-align="bottomright" />
